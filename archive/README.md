# archive

Files that have been superseded or deprecated. Nothing here is executed.

## Subfolders
- `deprecated/YYYY-MM-DD/` — files archived on a given date
- `old_versions/` — major version snapshots

## Rules
- Never delete from archive without explicit approval
- Files moved here get a note in docs/refactor_log.md
- Archive by date so it's clear when each file was retired
